### v0.0.1 - Initial Beta Release
**ShotKeeper** helps you organize your desktop screenshots instantly.
**What's New**
*   **Instant Drag & Drop**: Move files to folders immediately.
*   **Hotkeys**: Use 1-9 to sort files quickly.
*   **Multi-language**: English, Korean, French supported.
**Installation**
*   Download `.dmg` file below.
*   *Note: If you see a security warning, go to System Settings > Privacy & Security > Open Anyway.*
